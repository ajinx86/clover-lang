# clover-lang

A lightweight cross-platform programming language

## Building Clover

You will need the following tools to build Clover:

- A C17 compatible compiler (GCC 8+ or Clang 15+)
- Meson 1.7.0 or newer
- Ninja 1.12.1 or newer
